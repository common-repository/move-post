== Description ==

Move posts from one category to another.


== Installation ==

1. Upload 'move-post' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the Move Post 
4. You can choose from and to category.
5. Click on "Submit" button.
